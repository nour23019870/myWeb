Info

nour Writing is a computer vision project that allows users to write in the air using their finger. The application uses OpenCV and MediaPipe to track hand movements and translate them into digital text on screen. Users can draw letters, numbers, or simple shapes in the air, and the system recognizes and displays them in real-time. This contactless writing system offers an innovative way to interact with computers without physical input devices.

The Challenge

  • Accurately tracking fingertip movements through 3D space with a 2D camera
  • Differentiating between intentional writing gestures and unintended hand movements
  • Creating a responsive system that works in varying lighting conditions and backgrounds
  • Translating continuous hand motion into discrete characters or shapes

The Solution

  • Implemented MediaPipe's hand tracking solution to accurately detect hand landmarks
  • Created a gesture-based activation system using the distance between thumb and index finger
  • Used adaptive color thresholding techniques to maintain performance across various environments
  • Developed trajectory analysis algorithms to convert motion paths into recognizable characters
