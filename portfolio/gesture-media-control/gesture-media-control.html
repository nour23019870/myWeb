Info

A computer vision application that allows users to control media playback and system volume using hand gestures. Built with Python, OpenCV, and MediaPipe, the application provides intuitive gesture controls for common media tasks, making it possible to interact with your computer without touching a keyboard or mouse. The system features multiple control modes including volume control, media playback control, and a locked mode to prevent accidental inputs.

The Challenge

  • Creating reliable hand gesture recognition that works in varied lighting conditions
  • Designing intuitive gesture mappings that are easy to remember and perform
  • Integrating with system audio and media controls seamlessly

The Solution

  • Utilized MediaPipe's robust hand tracking system for accurate gesture detection
  • Implemented mode-switching functionality to expand the range of possible controls
  • Integrated PyCaw and system APIs for reliable media control with visual feedback
